# 朋友局德州扑克 ♠

一个和朋友在线联机玩的德州扑克网页游戏。纯虚拟筹码，带战绩排行榜，纯娱乐用途。

## 怎么玩

1. 打开游戏网址，输入昵称，点「创建新牌桌」，会得到一个 4 位房间号
2. 把网址和房间号发给朋友（或直接发 `网址/?room=房间号` 的链接）
3. 朋友输入昵称和房间号点「加入朋友的牌桌」
4. 人齐后（至少 2 人）房主点「开始发牌」，之后每局自动连续进行

## 游戏规则

- 每人入场自动带入 1000 虚拟筹码，盲注 5/10
- 标准无限注德州扑克：可弃牌、过牌、跟注、加注、全押，支持边池
- 轮到你时有 45 秒时间行动，超时自动过牌/弃牌
- 筹码输光后可以点「补充筹码」再拿 1000（会计入战绩的买入次数）
- 排行榜按「总盈亏」排名 = 当前筹码 −（买入次数 × 1000）
- 手机浏览器/微信里打开即可，无需安装；刷新页面或断线后重新加入同一房间会自动回到原座位和筹码

## 部署上线（给帮忙部署的人看）

这是一个 Node.js 应用，一条命令启动：

```bash
npm install
npm start        # 默认端口 3000，可用环境变量 PORT 修改
```

- 需要支持 WebSocket 的托管环境（Zeabur、Render、Railway、Koyeb、任意云服务器均可）
- 战绩保存在同目录的 `stats.json`（重启不丢；托管平台若无持久磁盘，重新部署后排行榜会清零）
- 无需数据库、无需其他配置

## 文件说明

- `server.js` — 服务器：房间管理、下注流程、结算、战绩
- `engine.js` — 扑克引擎：发牌、比牌、边池计算
- `public/index.html` — 游戏界面（手机适配）
- `test_engine.js` — 引擎自动化测试（含 2000 局随机模拟）
- `test_ui.js` — 浏览器联机流程自动化测试
